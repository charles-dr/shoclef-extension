# Extension - Shoclef Scraper

This extension scraps product info like title, description from various shopping site.


## ID
hakobpmphaleegackblhmmigplnlbndp

## storage structure

```json
{
  "websites": [],
  "products": [{
    "title": "",
    "description": "",
    "images": [],
    "variants": [],
    "brand": [],
    "price": 0,
    "oldPrice": 0,
    "currency": "USD"
  }],
  "scrapingURLs": [{
    "url": "",
    "scraped": false,
    "storedOnline": false
  }]
}
```

## Summary
### every website scrapper must have the following functions.

- initInfo
- doScrap
